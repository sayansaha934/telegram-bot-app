# How To Run Backend
- ```cd backend```
- Run ```npm install```
- Create a .env file and add the values
   ```
   TELEGRAM_API_ID=
   TELEGRAM_API_HASH=
   ```
- Run ```npm start```

# How To Run Frontend
- ```cd frontend```
- Run ```npm install```
- Create a .env file and add the values
   ```
   NEXT_PUBLIC_BASE_URL=http://localhost:3001/
   ```
- Run ```npm run dev```
